Enterprise Features

The following provide extra functionality to the core EasyFlow application. Different features are availiable with different packages.

Advanced Dashboards

Usually for a system like EasyFlow it is good to have some sort of way of measuring progress and efficiency within the various workflows and different users. Our advanced dashboards provide the users with a way of achieving that. Dashboards are customizable and can report on any data within the system.

Case Archive & Restore

This allows users to archive multiple cases quickly and efficiently, which enables a faster and more efficient system. It will remove all cases data from the various tables and store it. Cases can be restored any time.

Batch Routing

Provides functionality to move multiple cases to the next task and user simultaneously.

Data Reporting Tools

This is our brand new reporting wizard which allows users to create their own custom reports with built-in permissions. The reporting tool can perform joins on multiple data sets and any of the EasyFlow report tables, and export them to excel. This is all provided through an interactive UI wizard.
